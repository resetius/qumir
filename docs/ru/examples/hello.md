# Hello World

Простейшая программа — вывод текста на экран.

## Разбор

Любая программа на КуМире начинается с ключевого слова `алг` (алгоритм) и заключается между `нач` (начало) и `кон` (конец):

```
алг
нач
    ...
кон
```

Чтобы вывести текст на экран, используем команду `вывод`. Текст пишется в кавычках, а `нс` переводит курсор на новую строку:

```
    вывод "Привет, мир!", нс
```

## Полная программа

```
алг
нач
    вывод "Привет, мир!", нс
кон
```
