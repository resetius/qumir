# Метод GMRES

Обобщённый метод минимальных невязок (Generalized Minimal Residual) — итерационный метод решения СЛАУ **Ax = b** для произвольных (в том числе несимметричных) матриц.

## Ключевые понятия

- **Подпространство Крылова** — K_m = span{r₀, Ar₀, A²r₀, ..., A^(m-1)r₀}
- **Процесс Арнольди** — построение ортонормированного базиса подпространства Крылова
- **Вращения Гивенса** — ортогональные преобразования для решения задачи наименьших квадратов
- **Рестарты** — периодический сброс подпространства для экономии памяти

## Задача

Решается система с матрицей Гильберта. В отличие от метода CG, GMRES не требует симметричности и положительной определённости матрицы.

## Структура алгоритма

### 1. Внешний цикл (рестарты)

GMRES строит подпространство Крылова размерности m. Если решение не найдено за m шагов, выполняется рестарт: текущее приближение становится начальным, и процесс повторяется.

### 2. Процесс Арнольди (внутренний цикл)

На каждом шаге k:
- Вычисляем w = A × v_k
- Ортогонализуем w относительно всех предыдущих базисных векторов (по Грама-Шмидту)
- Нормируем и добавляем в базис

Результат — ортонормированный базис V и верхняя матрица Хессенберга H.

### 3. Вращения Гивенса

На каждом шаге применяем вращения Гивенса к матрице H для приведения к верхнетреугольному виду. Это позволяет следить за нормой невязки без явного вычисления приближения x.

### 4. Обратная подстановка и обновление x

Решаем треугольную систему Hy = g, затем обновляем приближение: x := x₀ + V·y.

## Сравнение с CG

| Свойство | CG | GMRES |
|---|---|---|
| Требования к матрице | СПД | Любая невырожденная |
| Память | O(n) | O(m·n) |
| Сходимость | ≤ n итераций | Зависит от спектра |
| Рестарты | Не нужны | Необходимы |

## Сложность

Каждая итерация внутреннего цикла стоит **O(n² + k·n)**, где k — номер итерации (из-за ортогонализации). Рестарт каждые m шагов ограничивает затраты на ортогонализацию.
