# Дом

Рисование домика с помощью Чертежника. Программа показывает, как строить фигуры по координатам, менять цвет линий и добавлять текстовые надписи.

## Разбор

### Подключение Чертежника

```
использовать Чертежник
```

Команда `использовать Чертежник` подключает исполнителя Чертежник, который умеет рисовать линии, перемещаясь по координатной плоскости.

### Основание дома

```
опустить перо
выбрать чернила(1)  | синий
сместиться в точку(0.0, 0.0)
сместиться в точку(4.0, 0.0)
сместиться в точку(4.0, 3.0)
сместиться в точку(0.0, 3.0)
сместиться в точку(0.0, 0.0)
```

Команда `опустить перо` включает рисование -- при каждом перемещении за пером остаётся линия. Команда `выбрать чернила(1)` задаёт синий цвет. Затем `сместиться в точку(x, y)` последовательно перемещает перо по четырём углам прямоугольника $4 \times 3$ и возвращает его в начало.

### Крыша

```
выбрать чернила(4)  | красный
сместиться в точку(0.0, 3.0)
сместиться в точку(2.0, 5.0)
сместиться в точку(4.0, 3.0)
```

Треугольная крыша рисуется красным цветом. Перо перемещается от левого верхнего угла основания к вершине $(2, 5)$ и далее к правому верхнему углу.

### Дверь

```
выбрать чернила(8)  | серый
сместиться в точку(1.5, 0.0)
сместиться в точку(1.5, 1.5)
сместиться в точку(2.5, 1.5)
сместиться в точку(2.5, 0.0)
```

Серый прямоугольник двери размером $1 \times 1{,}5$ в нижней части стены. Перо не поднимается -- после крыши оно движется прямо к двери, и соединительная линия от $(4, 3)$ к $(1{,}5, 0)$ тоже будет нарисована.

### Окно

```
выбрать чернила(3)  | голубой
сместиться в точку(0.5, 1.8)
сместиться на вектор(0.8, 0.0)
сместиться на вектор(0.0, 0.8)
сместиться на вектор(-0.8, 0.0)
сместиться на вектор(0.0, -0.8)
```

Окно рисуется голубым цветом. Здесь вместо абсолютных координат используется команда `сместиться на вектор(dx, dy)`, которая перемещает перо относительно текущей позиции. Это удобно для рисования фигуры, которую легко описать через стороны: вправо на $0{,}8$, вверх на $0{,}8$, влево на $0{,}8$, вниз на $0{,}8$.

### Надпись

```
поднять перо
выбрать чернила(0)  | черный
сместиться в точку(0.5, -0.5)
написать(3.0, "ДОМ")
```

Команда `поднять перо` отключает рисование -- перо перемещается без следа. Затем `написать(3.0, "ДОМ")` выводит текст с размером шрифта $3{,}0$ в текущей позиции пера.
