# Спираль из квадратов

Вращающиеся квадраты с увеличивающимся размером создают красивый спиральный узор. Пример показывает, как вложенные циклы и небольшой поворот порождают сложную фигуру.

## Разбор

Подключаем Черепаху и задаём начальный размер квадрата:

```
использовать Черепаха
алг
нач
    вещ размер
    размер := 10
```

Внешний цикл повторяется 30 раз --- столько квадратов будет нарисовано. Внутренний цикл рисует один квадрат: 4 стороны с поворотом на $90°$:

```
    нц для i от 1 до 30
        нц для j от 1 до 4
            вперед(размер)
            вправо(90)
        кц
```

После каждого квадрата увеличиваем размер на 5 и поворачиваем черепаху на $15°$. Именно этот небольшой поворот создаёт спиральный эффект --- каждый следующий квадрат немного развёрнут относительно предыдущего:

```
        размер := размер + 5
        вправо(15)
    кц
кон
```

## Полная программа

```
использовать Черепаха
алг
нач
    вещ размер
    размер := 10
    нц для i от 1 до 30
        нц для j от 1 до 4
            вперед(размер)
            вправо(90)
        кц
        размер := размер + 5
        вправо(15)
    кц
кон
```
