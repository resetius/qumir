# Ввод и вывод

Чтение данных с клавиатуры и вывод результата.

## Разбор

Сначала объявляем переменную `имя` строкового типа — `лит` (литеральный, т.е. текстовый):

```
    лит имя
```

Выводим подсказку для пользователя:

```
    вывод "Как вас зовут? "
```

Команда `ввод` считывает то, что пользователь набрал на клавиатуре, и записывает в переменную:

```
    ввод имя
```

В `вывод` можно перечислить несколько значений через запятую — текст в кавычках, переменные и `нс` (новая строка):

```
    вывод "Привет, ", имя, "!", нс
```

## Полная программа

```
алг
нач
    лит имя
    вывод "Как вас зовут? "
    ввод имя
    вывод "Привет, ", имя, "!", нс
кон
```
