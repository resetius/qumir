# Сумма чисел

Ввод двух чисел и вывод их суммы.

## Разбор

Объявляем две целочисленных переменных — тип `цел` (целое число):

```
    цел a, b
```

Просим пользователя ввести числа:

```
    вывод "Введите два числа: "
```

Команда `ввод` может считать сразу несколько переменных — пользователь вводит их через пробел или на отдельных строках:

```
    ввод a, b
```

В `вывод` можно вставлять арифметические выражения — `a + b` вычислится автоматически:

```
    вывод a, " + ", b, " = ", a + b, нс
```

```

нач
    цел a, b
    вывод "Введите два числа: "
```
